# 实验室管理系统

## 执行实验记录计算投料总量命令

```java

1.根据实验记录的操作过程， 把实验过程中用到的所有物料汇总， 生成投料方案， 并和实验记录进行管理关系
	=10. 创建项目， 当前用户是创建者
	=11. 创建实验记录， 当前用户是创建者
	=9. 实验记录默认状态就是： 实验中
	=8. 项目状态， 刚添加的时候， 没有值就是筹划中
	=7. 投料实体中， 实际投料没写， 则把计划投料的值复制给实际投料
	=6. 填投料信息的时候， 必须有物料
	=4. 创建项目， 选择组长的时候， 要把组员也增加上
	=5. 项目的组长必填
	=5. 实验记录提请归档后， 需要制定审核人（实验对应的项目，项目管理的组长为审核人
	实验记录关联项目必填
	
```

	1. 添加物料基础信息的时候增加一个库存信息， 库存量为0， 低库存阈值0， 高库存阈值0， 库存状态为【零状态】
	2. 库存量每次修改的时候， 计算高库存， 还是低库存

   			出入库记录 对应 一个批次  ， 保存出入库记录的时候， 通过出入库对应的批次找到对应的物			料， 并找到库存信息， 库存和批次进行关联， 
  
订单和物料预定：
	 1. 物料预定的预订人为当前用户
